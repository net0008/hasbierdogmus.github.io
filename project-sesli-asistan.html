Eğitimde Sesli Asistan

Python ile geliştirilen, öğretmene sınıf içinde rehberlik eden dijital yol arkadaşı.

Python & Flask

Proje Hakkında

Eğitimde Sesli Asistan projesi, ders senaryolarının akışını kontrol etmek ve öğretmene materyal sunumunda yardımcı olmak amacıyla geliştirilmiştir. Python (Flask) altyapısı üzerinde çalışan sistem, Google Text-to-Speech (gTTS) kütüphanesini kullanarak metinleri doğal bir konuşma diline çevirir.

Amaç: Öğretmenin tahtaya veya bilgisayara bağımlılığını azaltarak, sınıf içi etkileşimi artırmak ve ders materyallerini sesli komutlarla yönetilebilir hale getirmek.
  • Flask Web Server: Yerel ağda çalışan hızlı ve hafif backend.
  • Metin Okuma (TTS): Ders notlarını anlık olarak sese çevirme.
  • Senaryo Modu: Önceden planlanmış ders akışını adım adım takip etme.
  • Web Arayüzü: Telefondan veya tabletten kontrol imkanı.

Kodun Kalbi: Python & Flask

Projenin arka planında çalışan temel app.py yapısının basitleştirilmiş hali.

app.py 
from flask import Flask, render_template, request
from gtts import gTTS
import os

app = Flask(__name__)

# Asistanın konuşma fonksiyonu
def konus(metin):
    tts = gTTS(text=metin, lang='tr')
    dosya_adi = "static/ses.mp3"
    tts.save(dosya_adi)
    # Ses dosyasını oynat (OS modülü ile)
    os.system("start " + dosya_adi)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/komut', methods=['POST'])
def komut_al():
    komut = request.form['mesaj']
    
    if "merhaba" in komut.lower():
        yanit = "Merhaba hocam, ders için hazırım."
        konus(yanit)
        return yanit
    
    elif "dersi başlat" in komut.lower():
        yanit = "Ders başlıyor. Bugünkü konumuz: Türkiye'nin İklimi."
        konus(yanit)
        return yanit

if __name__ == "__main__":
    app.run(debug=True)
Geliştirme Notu: Bu proje şu anda prototip aşamasındadır. İlerleyen versiyonlarda OpenAI API entegrasyonu ile öğrencilerin sorularına doğal dilde yanıt verebilen bir yapıya dönüştürülmesi hedeflenmektedir.
